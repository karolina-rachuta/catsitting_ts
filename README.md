## Tech Stack

-   React (including React Router, Context API, and Hooks: useState, useEffect, useRef, useContext, ScrollLink)
-   SCSS for styling
-   React Hook Form
- EmailJS
-   Responsive Web Design (RWD) to ensure usability across different screen sizes.
